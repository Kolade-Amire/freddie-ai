# Freddie AI

A backend pipeline to automate recruitment using Google Sheets, Drive, Gmail, and OpenAI.

## Prerequisites
- Node.js (>= 16.x)
- Docker (optional for containerization)
- Google API credentials (Sheets, Drive, Gmail)
- OpenAI API key
